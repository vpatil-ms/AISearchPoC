Documenting Performance

All performance discussions should be documented and records of each discussion should be stored in the employee's personnel file. These records can help guide you when making future employment decisions, such as promotions, transfers, or terminations. These records can also help support those decisions if they are ever challenged.

Generally, these records should include:

  • Employee name and title
  • Reviewer name and title
  • Date of performance discussion
  • Performance issue (if applicable) and specific examples
  • Action plan and timeline for improvement (if applicable)

For performance reviews:

  • Performance review period (dates for which the employee is being evaluated)
  • Performance metrics (include responsibilities from the job description, assigned goals, along with overall company objectives)
  • A well-explained rating system (e.g., exceptional, exceeds expectations, meets expectations, improvement needed, unsatisfactory)
  • A comment field for each metric
  • A section for overall comments
  • A section for future goals and action plans
  • Signature lines for the employee and supervisor

Employers should be consistent in using the same metrics for evaluating performance for similar positions.
